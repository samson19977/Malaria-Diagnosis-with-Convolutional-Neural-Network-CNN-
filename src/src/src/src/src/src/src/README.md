# Malaria Diagnosis with Convolutional Neural Network (CNN)

This project implements a deep learning model to classify red blood cell images as either parasitized (infected with malaria) or uninfected. The model is built using TensorFlow/Keras and achieves high accuracy on the NIH Malaria dataset.

## Dataset
The dataset used is the [NIH Malaria Dataset](https://ceb.nlm.nih.gov/repositories/malaria-datasets/) available on Kaggle. It contains 27,558 cell images (13,779 parasitized and 13,779 uninfected).
